# Task Manager

Task tracking web app .

## Files
- `index.html` - structure and UI
- `styles.css` - styling
- `app.js` - app logic with localStorage persistence

## Features
- Add tasks with name, description, and due date
- Mark tasks as done
- Delete tasks
- Filter by all/active/done
- Clear all done tasks
- Persistent data in browser localStorage
